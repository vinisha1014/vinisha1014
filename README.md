#### 🧠 MotorAssistEnv
**Closed-Loop DBS Reinforcement Learning Environment**

OpenEnv-compatible RL benchmark for adaptive Deep Brain Stimulation in Parkinson's patients. Calibrated against the peer-reviewed Fleming et al. (2023) biophysical model from *Journal of Neural Engineering*.

▸ **3-task curriculum** with continuous DBS action space (amplitude, pulse width, motor command)
▸ **8-component reward** balancing motor function, beta/tremor suppression, safety budget, and clinical smoothness
▸ Bilinear-interpolated entrainment surface from a **12×15 parameter sweep**
▸ Patient-profile dynamics modelling fatigue, recovery, side-effect sensitivity
▸ Deployed publicly on HF Spaces with deterministic graders

**RL Benchmark for Software Delivery Under Uncertainty**

Deterministic OpenEnv environment that frames project management as a sequential decision problem — agents make daily staffing, prioritization, and intervention calls under dependency graphs and burnout cascades.

▸ Benchmarked **Qwen 2.5-72B** across 3 difficulty tiers
▸ Baseline scores **0.91 / 0.65 / 0.30** on the 22-day, 6-disruption hard scenario — exposes a clear optimization frontier
